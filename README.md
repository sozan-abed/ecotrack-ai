# ♻️ EcoTrack AI

EcoTrack AI is an AI-powered smart waste sorting assistant built using Teachable Machine and TensorFlow.js.

The system helps users identify and sort waste into the correct recycling category using a live camera or uploaded images.

## Features

- 📷 Live camera waste detection
- 🖼 Upload image analysis
- ♻️ Plastic, paper, metal, food, and clean area detection
- 🌱 Environmental awareness support
- 📱 Mobile-friendly interface
- 🤖 AI-powered classification
- 🎯 Real-time prediction results

## Technologies Used

- Teachable Machine
- TensorFlow.js
- HTML
- CSS
- JavaScript

## Purpose

EcoTrack AI was created to promote recycling awareness and support smarter environmental education through artificial intelligence.
